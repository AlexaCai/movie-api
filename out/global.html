addFavoriteMovies()

Allow to POST favorite movies into user account. This endpoint allows to add favorite movies to a user list of favorite in the MongoDB database based on the user's username and movie ID, and returns it as a JSON response. Authentication using JWT (received by users upon successful login) is required to use this route and add a favorite movie to a user list of favorite.

POST new favorite movies endpoint characteristics:

-Endpoint URL: /users/:Username/ movies/:movieID
-Query parameter(s) (parameter(s) in the URL): :Username & :movieID
-HTTP method: POST
-Request body data format: N.A.
-Request body data example: N.A.
